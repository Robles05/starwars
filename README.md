# Star Wars Search App
This is a React app , where you can search for Star Wars characters , spaceships, planets.

## Built with
The app was built using the SWAPI Api - https://swapi.co/documentation and it was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

## Deployment
The site was deployed using [GitHub Pages](https://pages.github.com/) following the steps for React apps outlined in this [article](https://itnext.io/so-you-want-to-host-your-single-age-react-app-on-github-pages-a826ab01e48).
